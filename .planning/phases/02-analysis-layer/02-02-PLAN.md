**1. Create js/analysis.js** with these exact exports and functions:

```javascript
// js/analysis.js
// Analysis card overlay: loads matchups.json, renders side-by-side analysis cards.
// Imports from state.js only. NEVER imports from app.js or bracket.js.

import { state } from "./state.js";

let matchups = {};

export async function loadMatchups() {
  const data = await fetch("./data/matchups.json").then(r => r.json());
  matchups = data.matchups;
}

export function openAnalysisCard(slotId) { ... }
export function closeAnalysisCard() { ... }
export function initAnalysisHandlers() { ... }
```

`openAnalysisCard(slotId)`:
- Look up `matchups[slotId]`
- If not found, return (no-op)
- Check `matchup.team_top` and `matchup.team_bot`. If either is null, render placeholder HTML inside `#analysis-panel`: `<div class="card-header"><button class="analysis-close" aria-label="Close">&times;</button></div><div class="card-placeholder"><p>Pick both teams to see analysis</p></div>`
- If both teams present, look up `state.teams[matchup.team_top]` and `state.teams[matchup.team_bot]` for seed/name display
- Build card HTML using the `buildCardHTML(matchup, topTeam, botTeam)` function from 02-RESEARCH.md code examples. The HTML structure must include:
  - `.card-header` with close button, round/region label, and flags div
  - `.card-teams` grid with two `.team-col` divs, each containing: `.team-title` (seed badge + name), `.metrics` (3 metrics: AdjEM, Last 10, SOS), `.pros` ul (3 li items), `.cons` ul (3 li items)
  - `.card-recommendation` with confidence badge (class `confidence--high|medium|low`), pick team name, and reasoning paragraph
  - Upset alert flag: `<span class="flag flag--upset">UPSET ALERT</span>` if `matchup.upset_alert === true`
  - Contrarian flag: `<span class="flag flag--contrarian">CONTRARIAN</span>` if `matchup.contrarian === true`
- Set `panel.innerHTML = cardHTML`
- Add `.visible` class to `#analysis-overlay`

`closeAnalysisCard()`:
- Remove `.visible` class from `#analysis-overlay`

`initAnalysisHandlers()`:
- Escape key listener on `document` (keydown, check `e.key === "Escape"`)
- Backdrop click on `#analysis-overlay` (click, check `e.target === e.currentTarget`)
- Close button via event delegation on `#analysis-panel` (click, check `e.target.closest(".analysis-close")`)
- All three call `closeAnalysisCard()`
- Register ALL handlers ONCE. Do not re-register on each card open.
