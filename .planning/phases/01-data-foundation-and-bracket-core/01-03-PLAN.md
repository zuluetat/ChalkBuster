- state.firstFour[ffId].resolved = true
- state.firstFour[ffId].winner = winner
- Find the R64 slot that sources from this ff game:
  const affectedSlot = Object.values(state.slots).find(
    s => s.top?.ff_id === ffId || s.bot?.ff_id === ffId
  )

- Re-render the First Four panel: renderFirstFour()

**export function clearFirstFourResolution(ffId)** — clears a resolved game and cleans up bracket:
- state.firstFour[ffId].resolved = false
- state.firstFour[ffId].winner = null
- Find affectedSlot same way as above
- If affectedSlot:
  - delete state.picks[affectedSlot.id]
  - clearDownstream(affectedSlot.id)  // from state.js
  - updateSlotAndDownstream(affectedSlot.id)
- Re-render the First Four panel: renderFirstFour()

**Event delegation** — attach single click listener on #first-four-panel:
- On click, find element with data-action attribute
