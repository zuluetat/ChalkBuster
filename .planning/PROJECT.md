# ChalkBuster

## What This Is

ChalkBuster is a web-based March Madness bracket analysis tool that helps Tim and friends make data-informed picks with a focus on finding strategic upsets. It provides side-by-side matchup analysis, confidence-tiered recommendations, and contrarian opportunity flags — all designed for quick (≤2 min/matchup) decision-making in a 6-15 person bracket pool with standard ESPN scoring.

## Core Value

Tim can complete his entire bracket using data-driven analysis that surfaces non-obvious upset picks to differentiate from competitors — ready before the Wednesday March 19 deadline.
